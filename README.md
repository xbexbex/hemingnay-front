# Hemingnay front
The front-end for [Hemingnay](https://github.com/xbexbex/hemingnay), an extremely bad fanfiction generator.

### Using development server

```shell
npm start
```

### Building

```shell
npm run build
```

### Type-checking

```shell
npm run type-check
```

### Using docker
```shell
docker build . -t <name-for-the-project>
docker run <name-for-the-project>
```
